### 文章内容字段定义表
| 字段名        | 数据类型      | 必填 | 描述                                                         |
| :------------ | :------------ | :--- | :----------------------------------------------------------- |
| **blocks**    | Array         | 是   | **根字段**。是一个数组，作为容器按顺序存放文章解析出的所有内容块对象。 |
| **type**      | String        | 是   | **内容类型**。用于标识当前块的性质，枚举值为 `heading`（标题）或 `paragraph`（段落）。 |
| **level**     | Integer       | 条件 | **标题层级**。仅当 `type` 为 `heading` 时存在。数值越小层级越高（例如 1 为主标题，2 为副标题）。 |
| **text**      | String        | 条件 | **标题文本**。仅当 `type` 为 `heading` 时存在，存储标题的原始字符串内容。 |
| **sentences** | Array[String] | 条件 | **句子列表**。仅当 `type` 为 `paragraph` 时存在。存储段落拆分后的句子数组，保留句末标点。 |
