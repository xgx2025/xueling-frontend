# 翻译功能快速启动指南

## 🚀 5分钟快速上手

### 第一步：查看翻译按钮

启动项目后，在页面右下角你会看到两个悬浮按钮：

```
┌─────────────────────┐
│                     │
│                     │
│                     │
│              🤖 ← AI助手 (紫色)
│              📄 ← 翻译助手 (绿色)
└─────────────────────┘
```

### 第二步：打开翻译面板

**方式一：点击按钮**
- 点击绿色的翻译按钮

**方式二：使用快捷键**
- Windows: `Ctrl + Shift + T`
- Mac: `Cmd + Shift + T`

### 第三步：翻译单词

1. 在输入框中输入单词，例如：`apple`
2. 按 `Enter` 或点击"翻译"按钮
3. 查看翻译结果：
   - 音标：/ˈæpl/
   - 释义：n. 苹果；苹果树
   - 发音：点击 🎤 按钮听发音

### 第四步：添加到单词本

1. 点击"添加到单词本"按钮
2. 选择目标单词本
3. 点击"确认添加"
4. 完成！单词已保存到你的单词本

## 💡 实用技巧

### 技巧1：快速复制
翻译结果后，点击"复制"按钮，可以快速复制单词和释义到剪贴板。

### 技巧2：语音学习
点击发音按钮，可以听到标准的英语发音，帮助你学习正确的读音。

### 技巧3：历史记录
翻译过的单词会自动保存，下次可以快速查看，无需重新翻译。

### 技巧4：快捷键
记住 `Ctrl+Shift+T`，随时随地快速打开翻译面板。

### 技巧5：单词本管理
翻译后直接添加到单词本，方便后续复习和记忆。

## ❓ 常见问题

### Q1: 翻译按钮在哪里？
**A:** 在页面右下角，绿色的圆形按钮，位于 AI 助手按钮下方。

### Q2: 快捷键不起作用？
**A:** 确保：
- 输入法切换到英文
- 没有其他应用占用该快捷键
- 浏览器窗口处于激活状态

### Q3: 发音功能不工作？
**A:** 发音功能依赖浏览器的 Web Speech API：
- 推荐使用 Chrome 或 Edge 浏览器
- 确保浏览器允许音频播放
- 检查系统音量设置

### Q4: 历史记录丢失了？
**A:** 历史记录存储在浏览器的 localStorage 中：
- 清除浏览器数据会删除历史
- 无痕模式不会保存历史
- 不同浏览器的历史是独立的

### Q5: 如何清空历史记录？
**A:** 在翻译面板底部的"最近翻译"区域，点击"清空"按钮。

### Q6: 可以翻译句子吗？
**A:** 目前主要支持单词翻译，句子翻译功能即将推出。

### Q7: 添加到单词本失败？
**A:** 检查：
- 是否已登录
- 网络连接是否正常
- 是否已创建单词本
